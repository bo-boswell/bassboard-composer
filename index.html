Bassboard Composer

Turn the notes you play into a Nashville Number chart

Click the notes on the fretboard and Bassboard Composer turns them into a printable number chart. Right now, clicking a string just plays the note — start a chart to turn your clicks into chord numbers.

How it works
  1. Set your key, meter, and tuning at the top.
  2. Click Start a Chart, then click fretboard notes — each one becomes a number.
  3. Drag on the beat grid to set how long a chord lasts; add quality (minor, 7th…), a slash note, or a measure note.
  4. Use the ＋ on a section (or ＋ Add section at the bottom) to add sections like Verse and Chorus; click a section name to rename it.
  5. The File menu holds Save, Load, Copy, View Chart, and Clear. Save downloads a backup file; View Chart opens a printable page.
